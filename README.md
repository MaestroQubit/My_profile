# Chirag Kumar - Portfolio Website

A modern, responsive portfolio website showcasing my skills, projects, and achievements as a Computer Science Engineering student and aspiring web developer.

View the website here:- https://my-profile-taupe-gamma.vercel.app/

## 🚀 Features

- **Responsive Design**: Optimized for all devices and screen sizes
- **Modern UI/UX**: Clean, professional design with smooth animations
- **Contact Form**: Functional contact form with backend integration
- **Skills Showcase**: Interactive progress bars and skill categories
- **Project Portfolio**: Dedicated section for showcasing projects
- **Education & Achievements**: Academic background and accomplishments

## 🛠️ Tech Stack

### Frontend
- **React.js** - UI library
- **Tailwind CSS** - Styling framework
- **Lucide React** - Icon library
- **Axios** - HTTP client for API calls

### Backend
- **FastAPI** - Python web framework
- **MongoDB** - Database for storing contact messages
- **Pydantic** - Data validation
- **Motor** - Async MongoDB driver

## 🏃‍♂️ Getting Started

### Prerequisites
- Node.js (v16 or higher)
- Python (v3.8 or higher)
- MongoDB

### Frontend Setup
```bash
cd frontend
npm install
npm start
```

### Backend Setup
```bash
cd backend
pip install -r requirements.txt
uvicorn server:app --reload --host 0.0.0.0 --port 8001
```

## 🔧 Customization

The portfolio is designed to be easily customizable:
- Update personal information in `/frontend/src/data/mock.js`
- Modify color scheme in Tailwind CSS classes
- Add new sections by creating components
- Extend backend API for additional functionality

- Desktop (1920px and above)
- Tablet (768px - 1199px)
